Bitcoin halving is a term used to describe the point in which bitcoin network rewards to miners is reduced by half. The reward, which consists of new bitcoins (distributed by the network), will be half what it was at its latest block reward. This event occurs on approximately every 210,000 blocks or roughly every four years. Next halving at block 840,000
